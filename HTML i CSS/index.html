THE DIV TAG

  • Div stands for 'division'
  • Used to divide content into logical sections


ID's and Classes

  • An ID is a UNIQUE identifier, used once on a page.
  • A Class is an identifier of an element which can be used multiple times on a page.
  • Both are used for JS and CSS.
  • Example: div id=header
  • div class znajduje się jakby wewnątrz div id. Może być wykorzystany wielokrotnie w div id.
